# Django Polls App

A feature-rich polling application built with Django, extending the official tutorial with custom functionality like multi‑question voting, intuitive navigation, and a persistent background image.

## Features
- **Single Question Voting** – Vote on individual questions with immediate results.
- **Multi‑Question Voting** – Vote on all questions at once from a single page.
- **Results Pages** – View results for one question (with previous/next navigation) or all questions together.
- **Jump to Any Question** – From any results page, see a list of all questions with links to vote or view results.
- **Custom Background** – A consistent background image (`Background.jpg`) across all pages.
- **Admin Interface** – Easily manage questions and choices via Django's built‑in admin.

## Prerequisites
- Python 3.8 or higher
- pip (Python package manager)
- Git (to clone the repository)
